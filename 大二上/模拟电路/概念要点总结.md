3. **滤波电路**

   将整流电路输出的单向脉动电压中的交流成分滤掉，输出比较平滑的直流电压；

   - **性能比较**

     **电容**滤波适合“高压**小电流**”，**电感**滤波适合“低压**大电流**”。

     |     **比较维度**      |                        **电容滤波 **                         |                    **电感滤波**                     |
     | :-------------------: | :----------------------------------------------------------: | :-------------------------------------------------: |
     | **输出电压 ($U_o$)**  | **较高**   $\approx 1.2 U_2$ (空载时可达 $\approx 1.41 U_2$) | **较低**   $\approx 0.9 U_2$ (类似未滤波时的平均值) |
     | **外特性 (带载能力)** |          **差** (软特性)   负载增加时，电压下降较快          |   **好** (硬特性)   负载增加时，电压基本保持稳定    |
     |  **二极管电流冲击**   |                    **很大** (有浪涌电流)                     |                **小** (电流变化平缓)                |
     |     **典型应用**      |                  小功率电子设备、前置放大器                  |          大功率电源、焊接设备、雷达发射机           |

4. **稳压电路**

   利用自动调整的原理，使得输出直流电压稳定。

# 场效应管

只有一种载流子参与导电，且利用电场效应来控制电流的三极管，称为场效应管，也称单极型三极管。

- **分类**

  - **结型场效应管**
  - 绝缘栅场效应管


> 结型场效应管

- **结构**

  包含三个电极：

  - **栅极 (Gate, G)**：控制端；

  - **源极 (Source, S)**：电子（或空穴）的源头；

  - **漏极 (Drain, D)**：电子收集端。

- **栅源电压 $U_{GS}$​**

  - **电压极性：** 为了保证高输入电阻并实现控制，栅极与源极之间的PN结必须处于**反向偏置**状态。

  - **控制机理** | 耗尽层效应
    - 当 $U_{GS} = 0$ 时，沟道最宽，导电能力最强。

    - 随着 $|U_{GS}|$ 增大（即负得更多），PN结的**耗尽层加宽**。

    - 耗尽层向沟道中心延伸，导致导电沟道变窄，等效电阻变大，从而抑制漏极电流 $I_D$。

    - **夹断电压$U_P$：** 当 $U_{GS}$ 负到一定程度时，两侧耗尽层在沟道合拢，沟道被完全夹断，此时 $I_D \approx 0$​。

- **漏源电压 $U_{DS}$​**

  - **电压极性：** $U_{DS}$ 提供电场，使源极的载流子流向漏极。对于N沟道，$U_{DS} > 0$。
  - **非均匀影响：** $U_{DS}$ 沿沟道产生电位降。靠近漏极（D）端的PN结反向偏置电压最大，因此漏极端的耗尽层比源极端的更厚，沟道呈现**楔形**。


> **双极型三极管和场效应管性能比较**

| 比较项目 | 双极型三极管 | 场效应管 |
| :--: | :--: | :--: |
| **导电载流子** | **多子和少子**同时参与导电 (双极型) | 仅**多子**参与导电 (单极型) |
| **控制机制** | **电流控制电流器件**($i_B$ 控制 $i_C$) | **电压控制电流器件 **($u_{GS}$ 控制 $i_D$) |
| **输入电阻** | **较低** | **很高** |
| **温度稳定性** | **较差**<br>(少子对温度敏感，$\beta$ 和 $I_{CEO}$ 随温度变化大) | **较好**<br>(多子导电，受温度影响较小) |
| **噪声系数** | 较大 | 较小 (载流子主要在体内运动，表面效应小) |
| **抗辐射能力** | 较差 | 较强 |
| **制造工艺与集成度** | 工艺复杂，占用芯片面积较大 | 工艺简单，占用芯片面积小，适合大规模集成 |
| **零点漂移** | 较大 | 较小 (适合做直流放大器的输入级) |
| **静电敏感度** | 不敏感 | **敏感** |
