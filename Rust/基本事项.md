# 常用命令速查

- **为 rust-analyzer 配置关联项目**

    手动告诉 rust-analyzer 哪里能找到 Cargo.toml。在`.vscode/settings.json` 下进行管理：
    ```
    {
        "rust-analyzer.linkedProjects": [
            "Rust/hello_rust/Cargo.toml"
        ]
    }
    ```
- **使用 cargo 创建、构建、运行项目**

    以下操作需要确保在项目根目录下执行。

    1. 创建
        ```
        cargo new <project>
        ```
    2. 构建
        ```
        cargo build
        ```
    3. 运行
        ```
        cargo run
        ```
    4. 检查
        
        用于快速确认是否存在编译错误，无需生成可执行文件。
        ```
        cargo check
        ```

# 学习计划

## 第一阶段：环境与基础语法

**目标：能写出可运行的简单程序**

- 安装 Rust（rustup、cargo、rustc）
- 项目结构：`Cargo.toml`、`src/main.rs`
- 基本数据类型
  - 整数、浮点、布尔、字符
  - 元组、数组
- 变量与常量（`let`、`const`、`mut`）
- 控制流
  - `if / else if / else`
  - `loop / while / for`
  - `match` 基础用法
- 函数定义与返回值
- 注释与文档注释

---

## 第二阶段：所有权系统（Rust 核心）

**目标：理解 Rust 最独特的内存模型**

1. **所有权（Ownership）**
   - 每个值只有一个所有者
   - 值离开作用域时自动释放
2. **移动（Move）与复制（Copy）**
   - `Copy` trait 的类型
   - 所有权转移的场景
3. **借用（Borrowing）**
   - 不可变引用 `&T`
   - 可变引用 `&mut T`
   - 同一时刻只允许一个可变引用
4. **生命周期（Lifetime）基础**
   - 悬垂引用问题
   - 编译器如何检查引用有效性

---

## 第三阶段：核心数据结构与标准库

**目标：掌握日常编程的常用工具**

- **字符串**：`String` vs `&str` 的区别与转换
- **集合类型**
  - `Vec<T>`
  - `HashMap<K, V>`
  - `HashSet<T>`
- **结构体（Struct）**
  - 普通结构体、元组结构体
  - 方法（`impl` 块）
- **枚举（Enum）**
  - 带数据的枚举
  - `Option<T>`
  - `Result<T, E>`
- **模式匹配深入**
  - 解构、守卫、`@` 绑定

---

## 第四阶段：错误处理

**目标：写出健壮的程序**

- `panic!` 与不可恢复错误
- `Result` 与可恢复错误
- `?` 操作符传播错误
- 自定义错误类型
- `thiserror` / `anyhow`（生态常用库）

---

## 第五阶段：Trait 与泛型

**目标：理解 Rust 的抽象机制**

- **泛型**
  - 泛型函数、泛型结构体
  - 单态化原理
- **Trait**
  - 定义与实现 trait
  - 默认方法
  - 常用标准 trait：`Display`、`Debug`、`Clone`、`Iterator`
- **Trait bounds**
  - `T: Trait` 语法
  - `where` 子句
- **Trait 对象**
  - `dyn Trait`（动态分发）
  - 与泛型的对比

---

## 第六阶段：模块与包管理

**目标：组织中等规模的项目**

- 模块系统：`mod`、`use`、`pub`
- 文件与目录的模块映射
- Crate 类型：binary vs library
- Cargo 依赖管理
  - `Cargo.toml` 特性标志（features）
  - workspace 多包管理
- 文档测试（`///` + `cargo test`）

---

## 第七阶段：函数式编程特性

**目标：写出简洁、表达力强的代码**

- 闭包（Closure）
  - 捕获环境的三种方式：`Fn`、`FnMut`、`FnOnce`
- 迭代器（Iterator）
  - `map`、`filter`、`fold`、`chain`、`zip`
  - 自定义迭代器（实现 `Iterator` trait）
- 惰性求值与零成本抽象

---

## 第八阶段：智能指针与内存进阶

**目标：处理复杂的数据结构和所有权场景**

- `Box<T>`：堆分配
- `Rc<T>` / `Arc<T>`：引用计数
- `RefCell<T>` / `Mutex<T>`：内部可变性
- 常见组合：`Arc<Mutex<T>>`
- 循环引用与 `Weak<T>`

---

## 第九阶段：并发与异步

**目标：充分利用多核性能**

**同步并发**
- 线程：`std::thread::spawn`
- 消息传递：`mpsc` channel
- 共享状态：`Mutex`、`RwLock`

**异步编程**
1. `async` / `await` 语法
2. Future 模型与执行器原理
3. 选择异步运行时：`tokio` 或 `async-std`
4. 异步 I/O、并发任务：`tokio::spawn`、`join!`

---

## 第十阶段：高级特性

**目标：读懂复杂代码，写出底层库**

- 生命周期高级用法
  - 结构体中的生命周期标注
  - `'static` 的含义
- 高级 Trait
  - 关联类型（`type Item`）
  - 运算符重载（`std::ops`）
  - `Deref` 强制转换
- 宏（Macro）
  - 声明宏：`macro_rules!`
  - 过程宏概览：`derive`、`attribute`、`function-like`
- `unsafe` Rust
  - 使用场景与边界
  - 裸指针、FFI 基础

---

## 第十一阶段：实战项目

**目标：将所有知识融入真实场景**

- CLI 工具（推荐：`clap` 库）
- 网络服务（推荐：`axum` 或 `actix-web`）
- 并发爬虫或数据处理管道
- 实现一个小型数据结构库（链表、图等）
