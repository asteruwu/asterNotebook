# 基础准备与数据初始化

## 生成向量

- **生成方式**
  - **冒号运算符：**已知步长时使用。
  - **`linspace`命令：**已知“总点数”时使用（如在区间内取 100 个点）。

- **示例代码：**

  ```matlab
  % --- 方法1：冒号运算符 (起点:步长:终点) ---
  % 产生 0, 0.1, 0.2 ... 10
  t = 0:0.1:10;           % 最常用的方式
  
  % --- 方法2：linspace (起点, 终点, 点数) ---
  % 产生 0 到 2*pi 之间的 100 个均匀点
  x = linspace(0, 2*pi, 100); 
  
  % --- 方法3：手动输入 ---
  % 用于数据量少且不规则的情况
  v = [0, 0.2, 0.4, 0.6];
  ```

## 生成矩阵与空间分配

- **生成方式**

  - 避免在循环中动态扩容数组，先用 `zeros` 分配好空间。
  - **随机数**：区分均匀分布 (`rand`) 和正态分布 (`randn`)。

- **示例代码：**

  ```matlab
  n = 100; m = 3;
  
  % --- 1. 特殊矩阵初始化 ---
  A = zeros(n, m);        % 生成 n行m列 的全0矩阵 (预分配首选)
  B = ones(n, m);         % 生成全1矩阵
  I = eye(n);             % 生成 n阶 单位矩阵 (对角线为1)
  
  % --- 2. 随机数生成 ---
  R1 = rand(5, 3);        % 0到1之间的均匀分布随机数 
  R2 = randn(1000, 1);    % 标准正态分布随机数 (均值0, 方差1) 
  
  % --- 3. 复数矩阵输入 ---
  C = [1+1i, 2; 3i, 4];   % 直接输入复数 
  ```

## 生成网格数据

常用于3D/等高线绘图，在绘制 $z=f(x,y)$ 之前，必须将单维的 $x, y$ 向量转换为网格矩阵。

- **生成方式**

  - 使用 `meshgrid`。

- **示例代码：**

  ```matlab
  x = -2:0.1:2;           % x轴向量
  y = -1:0.1:1;           % y轴向量
  
  % --- 生成网格 ---
  % X, Y 都是矩阵，用于覆盖整个平面区域
  [X, Y] = meshgrid(x, y); 
  
  % --- 计算高度 Z ---
  % 必须使用点运算 (.^, .*, ./)
  Z = X.^2 + Y.^2;
  ```

# 数值计算与线性代数

## 矩阵性质分析与验证

- **考法：**给出一个具体的矩阵（可能包含复数），要求计算**秩、行列式、迹、特征值**等，并**验证**数学定理（如：矩阵的迹等于特征值之和）。

- **解题步骤：**

  1. **定义矩阵**：准确输入矩阵，注意复数用 `i` 或 `j` 表示，行与行之间用分号隔开；
  2. **调用函数**：使用对应的指令（如 `rank`, `det`）计算指标；
  3. **验证恒等式**：
     - 不要直接用 `==` 判断（因为浮点数存在误差）。
     - **核心技巧**：计算两边差值的范数或绝对值，判断是否小于极小值（如 `1e-5`）。
       - *例*：验证 $AX = X\Lambda$​，应检查 `norm(A*V - V*D) < 1e-5`。

- **示例代码：**

  ```matlab
  A = [...];                  % 定义矩阵 (注意分号换行)
  
  % --- 1. 计算属性 ---
  r = rank(A);                % 矩阵的秩
  d = det(A);                 % 行列式
  t = trace(A);               % 迹 (对角线元素之和)
  
  % --- 2. 特征值分解 ---
  [V, D] = eig(A);            % V:特征向量矩阵, D:特征值对角阵
  lambda = diag(D);           % 提取特征值向量
  
  % --- 3. 验证定理 (重点) ---
  % 验证 迹 == 特征值之和
  % abs()用于标量，norm()用于矩阵/向量
  check_trace = abs(t - sum(lambda)) < 1e-5; 
  
  % 验证特征方程 AX = X*Lambda
  LHS = A * V;                % 等式左边
  RHS = V * D;                % 等式右边
  check_eig = norm(LHS - RHS) < 1e-5; % 验证差值范数是否趋近于0
  ```

## 多项式拟合与求解

- **考法：**给定一组数据点 $(x, y)$，要求用指定阶数的多项式拟合，或者构建范德蒙矩阵求解线性方程组来获得系数。

- **解题步骤：**

  这本质上是求解线性方程组 $Xa = y$。

  1. **构建矩阵：**根据拟合公式构建范德蒙矩阵，即每一列是 $x$​ 的不同次幂。
  2. **求解系数：**使用 **左除运算符 (`\`)** 求解。
  3. **对比验证**：调用 `polyfit`，注意其输出系数通常是**降幂排列**，需注意顺序对比。

- **示例代码：**

  ```matlab
  x = [...]; y = [...];       % 定义数据点 (x需为列向量)
  
  % --- 1. 构建系数矩阵 (范德蒙矩阵) ---
  % 假设拟合 n 阶，构造 x^0 到 x^n 的各列
  X_mat = [ones(size(x)), x, x.^2, ...]; 
  
  % --- 2. 求解线性方程组 ---
  % 求解 X_mat * a = y
  a_linear = X_mat \ y;       % 左除求解，数值最稳定
  
  % --- 3. 内置函数对比 ---
  a_poly = polyfit(x, y, n);  % n为拟合阶数
  % 注意：a_poly 通常是降幂排列，与 a_linear 顺序可能相反
  ```

## 多项式乘法与卷积

- **考法：**求两个多项式的乘积。

- **解题步骤：**利用多项式乘法等价于**卷积**的原理，直接调用函数。

- **示例代码：**

  ```matlab

  w = conv(u, v);             % 计算卷积，即多项式乘积
  ```

# 符号数学计算

## 求极限与导数

- **考法：**给定一个分段函数，要求求某点的左极限（或左导数）和右极限（或右导数），通过判断两者是否相等来确定函数在该点是否连续或可导。

- **解题步骤：**

  1. **定义符号：**必须先用 `syms` 声明变量。
  2. **定义函数：**若是分段函数，使用 `piecewise`。
  3. **计算极限：**利用导数定义 $\lim_{h \to 0} \frac{f(x+h)-f(x)}{h}$，配合 `limit` 指令的 `'left'` 和 `'right'` 参数。
  4. **判断逻辑：**使用 `isequal` 比较左右结果。

- **示例代码：**

  ```matlab
  syms x h;                   % 定义符号变量
  f = piecewise(x>=0, x, x<0, -x); % 定义分段函数 (示例)
  
  % --- 1. 计算导数定义式的极限 ---
  % 导数定义: (f(x0+h) - f(x0)) / h
  term = (subs(f, x, 0+h) - subs(f, x, 0)) / h; 
  
  L = limit(term, h, 0, 'left');   % 求左导数
  R = limit(term, h, 0, 'right');  % 求右导数
  
  % --- 2. 判断可导性 ---
  check = isequal(L, R);      % 若相等则返回 1 (True)
  ```

## 泰勒级数展开

- **考法：**求函数在某点的 $n$ 次泰勒展开式，并与原函数绘图对比。

- **解题步骤：**

  1. **展开级数**：使用 `taylor` 指令。**注意**：`Order` 参数 $n$ 代表截断误差项 $O(x^n)$，生成的最高次项是 $x^{n-1}$。
  2. **可视化**：使用 `fplot`（符号绘图专用）在同一图上绘制原函数和近似多项式。

- **示例代码：**

  ```matlab
  syms x;
  f = sin(x);                 % 定义原函数
  
  % --- 1. 泰勒展开 ---
  % 'Order' 为 n 表示展开到 n-1 次幂
  T = taylor(f, x, 'Order', 6); 
  
  % --- 2. 绘图对比 ---
  figure; 
  fplot(f); hold on;          % 绘制原函数
  fplot(T);                   % 绘制展开式
  legend('Original', 'Taylor');
  ```

## 符号矩阵分析与恒等式验证

- **考法：**定义全是字母变量的矩阵，求行列式、迹、特征值，并验证代数恒等式（如 $tr(A) = \sum \lambda_i$）。

- **解题步骤：**

  1. **定义矩阵：**使用 `syms` 定义矩阵内的元素变量。
  2. **计算属性：**调用 `det`, `trace`, `eig`。
  3. **验证恒等式：**符号计算没有浮点误差，可以使用 `isAlways` 函数判断数学逻辑是否恒成立，或者判断 `simplify(左边 - 右边) == 0`。

- **示例代码：**

  ```matlab
  syms a b c d;               % 定义矩阵元素变量
  A = [a, b; c, d];           % 构建符号矩阵
  
  % --- 1. 计算矩阵指标 ---
  d_val = det(A);             % 行列式 (结果是表达式 ad-bc)
  t_val = trace(A);           % 迹
  lam = eig(A);               % 特征值向量
  
  % --- 2. 验证恒等式 ---
  % isAlways 判断符号逻辑是否恒为真
  check_trace = isAlways(t_val == sum(lam));
  ```

## 函数极值点分析

- **考法：**求函数的驻点（一阶导数为0的点），并利用二阶导数符号判断是极大值还是极小值。

- **解题步骤：**

  1. **求一阶导：**`diff(f)`。
  2. **解出驻点：**`solve(df == 0)`。
  3. **二阶导判别：**计算 `diff(f, 2)`，用 `subs` 代入驻点，判断正负（正为极小，负为极大）。

- **示例代码：**

  ```matlab
  syms x;
  f = ...;                    % 定义函数
  
  % --- 1. 求驻点 ---
  df = diff(f);               % 一阶导数
  x_sol = solve(df == 0, x);  % 解方程 f'(x)=0
  
  % --- 2. 二阶导数判别 ---
  d2f = diff(f, 2);           % 二阶导数
  val = subs(d2f, x, x_sol);  % 将驻点代入二阶导数计算数值
  
  % 根据 val 的正负即可判断 (正小负大)
  ```

## 求解微分方程

- **考法：**求解高阶微分方程（如 $y''' - y = 0$），带初始条件，并绘图。

- **解题步骤：**

  1. **定义方程：**使用 `diff(y, x, n)` 表示 $y$ 的 $n$ 阶导数。
  2. **定义条件：**如果有一阶导数初值 $y'(0)=0$，需先定义 `Dy = diff(y)`，然后写 `Dy(0)==0`。
  3. **求解：**用 `dsolve`。

- **示例代码：**

  ```matlab
  syms y(x);                  % 必须声明 y 是 x 的函数
  
  % --- 1. 定义方程与条件 ---
  eqn = diff(y, x, 2) == y;   % 二阶微分方程 y'' = y
  Dy = diff(y);               % 定义导数变量用于写初值
  cond = [y(0)==1, Dy(0)==0]; % 初始条件向量
  
  % --- 2. 求解与绘图 ---
  y_sol = dsolve(eqn, cond);  % 得到解析解
  fplot(y_sol, [0, 5]);       % 绘制解曲线
  ```

# 图形可视化处理

## 双Y轴绘图

- **考法：**当两组数据数值差异巨大或物理单位不同时，要求在同一张图中用左、右两个纵坐标轴分别展示。

- **解题步骤：**

  1. **准备数据：**准备好横坐标 $x$ 和两组纵坐标数据 $y_1, y_2$。
  2. **调用函数：**使用 `plotyy` 函数。
  3. **指定线型：**`plotyy` 可以指定左右轴分别使用什么绘图方式（如左边 `plot`，右边 `stem`）。

- **示例代码：**

  ```matlab
  x = ...; y1 = ...; y2 = ...; % 定义数据
  
  % --- 绘制双Y轴图 ---
  % fun1, fun2 分别指定左右轴的绘图函数 (如 'plot', 'stem')
  % ax: 坐标轴句柄 (ax(1)左, ax(2)右)
  % h1, h2: 图形对象句柄 (用于后续修改颜色/线型)
  [ax, h1, h2] = plotyy(x, y1, x, y2, 'plot', 'stem');
  ```

## 高维数据可视化 | 散点图

- **考法：**要求在二维平面上展示 4 维数据（坐标 $x, y$ + 大小 $Size$ + 颜色 $Color$）。

- **解题步骤：**

  1. **准备向量：**除了坐标 $x, y$，还需生成对应的大小的向量 $sz$ 和颜色的向量 $c$。
  2. **绘图：**调用 `scatter` 函数，并设置 `'filled'` 填充颜色。
  3. **辅助显示：**调用 `colorbar` 显示颜色映射条。

- **示例代码：**

  ```matlab
  x = ...; y = ...;       % 坐标数据
  sz = linspace(1,30,n);  % 大小映射向量
  c = linspace(1,10,n);   % 颜色映射向量
  
  % --- 绘制散点图 ---
  scatter(x, y, sz, c, 'filled'); 
  colorbar;               % 显示颜色条
  ```

## 统计图形与子图布局

- **考法：**要求在一个窗口中划分多个区域（子图），分别绘制直方图、堆积柱状图（垂直/水平）。

- **解题步骤：**

  1. **切分窗口：**使用 `subplot(行, 列, 序号)` 激活绘图区。
  2. **绘制直方图：**使用 `hist` 观察数据分布。
  3. **绘制柱状图：**使用 `bar`（垂直）或 `barh`（水平），配合 `'stacked'` 参数实现堆积效果。

- **示例代码：**

  ```matlab
  figure;                 % 新建窗口
  
  % --- 子图1: 直方图 ---
  subplot(2, 2, 1);       % 激活位置
  hist(data, 10);         % 分10个区间统计
  
  % --- 子图2: 垂直堆积柱状图 ---
  subplot(2, 2, 3);
  bar(data_mat, 'stacked');
  
  % --- 子图3: 水平堆积柱状图 ---
  subplot(2, 2, 4);
  barh(data_mat, 'stacked');
  ```

## 网格数据与等高线图

- **考法：**绘制二元函数 $z=f(x,y)$ 的等高线（2D/3D/填充）。

- **解题步骤：**

  1. **生成网格：**这是**必做步骤**。利用 `linspace` 生成单维向量，再用 `meshgrid` 生成网格矩阵 $X, Y$。
  2. **计算函数值：**计算 $Z = f(X, Y)$（注意使用**点运算**）。
  3. **绘图：**调用 `contour` (普通), `contourf` (填充), `contour3` (三维)。

- **示例代码：**

  ```matlab
  x = ...; y = ...;       % 单维向量
  [X, Y] = meshgrid(x,y); % 生成网格矩阵 (关键步骤)
  Z = sin(X) + cos(Y);    % 计算高度数据 (注意点运算)
  
  % --- 绘制等高线 ---
  contour(X, Y, Z, 20);   % 20 代表等高线条数
  % 或使用 contourf(填充), contour3(三维)
  ```

## 图形后期处理*

- **考法：**画完图后，通过代码修改图形属性（如坐标轴范围、对数刻度、线型、标记、图例等）。

- **解题步骤：**

  1. **获取句柄：**绘图时必须保存句柄，如 `ax = gca` 或 `h = plot(...)`。
  2. **修改属性：**使用 **`set(句柄, '属性名', 属性值)`** 指令。
  3. **常用属性：**需要记忆 `XDir` (反转), `YScale` (对数), `LineWidth` (线宽), `Marker` (标记) 等。

- **示例代码：**

  ```matlab
  % 获取句柄 (Handle)
  fig = figure;
  ax = gca;               % 获取当前坐标轴句柄
  h_line = plot(...);     % 获取曲线句柄
  
  % --- 修改对象属性 (set指令) ---
  % 调整坐标轴: 反向、对数刻度、显示网格
  set(ax, 'XDir', 'reverse', 'YScale', 'log', 'XGrid', 'on');
  
  % 调整线条: 线宽、虚线、标记点样式
  set(h_line, 'LineWidth', 2, 'LineStyle', '--', 'Marker', 'o');
  
  % --- 添加标注 ---
  title(ax, 'Title Name'); % 指定坐标轴添加标题
  legend([h1, h2], 'Name1', 'Name2'); % 指定对象添加图例
  ```

# 程序流程控制与函数句柄

## 流程控制 | 循环与判断

- **考法：**编写脚本计算数列（如斐波那契数列）、级数求和或进行迭代计算。通常要求结合 `for`/`while` 循环和 `if` 判断语句，有时包含 `break` 跳出逻辑。

- **解题步骤：**

  1. **初始化：**设置初始值（如 `f(1)=1`）或空向量。
  2. **循环结构：**根据是“固定次数”还是“满足条件”选择 `for` 或 `while`。
  3. **条件判断：**在循环内部使用 `if-else` 处理特殊逻辑或中断循环。
  4. **迭代更新：**确保循环变量或数组索引在变化。

- **示例代码：**

  ```matlab
  f = zeros(1, n);        % 预分配数组空间
  f(1) = 1;               % 初始化
  
  % --- 循环与判断 ---
  for i = 2:n             % 固定次数循环
      val = f(i-1) * 2;   % 迭代计算逻辑
  
      if val > threshold  % 条件判断
          break;          % 满足条件则跳出
      else
          f(i) = val;     % 存储结果
      end
  end
  ```

## 自定义函数 | m文件

- **考法：**要求编写一个独立的 `.m` 文件函数，能够接收输入参数，进行计算并返回输出参数（如计算圆的面积和周长，或阶乘函数）。

- **解题步骤：**

  1. **函数声明：**首行必须是 `function [out1, out2] = func_name(in1, in2)`。
  2. **输入检查：**检查输入参数是否合法（如 `r<0` 报错）。
  3. **计算逻辑：**编写核心公式。
  4. **返回结果：**将计算结果赋值给输出变量 `out1`, `out2`。

- **示例代码：**

  ```matlab
  % 定义函数: 输入 x, 返回 y1 和 y2
  function [y1, y2] = my_func(x)
  
      % 输入检查 (可选)
      if x < 0
          error('Input must be positive'); 
      end
  
      % 计算主体
      y1 = x .^ 2;        % 计算第一个输出
      y2 = x * 10;        % 计算第二个输出
  end
  ```

## 函数句柄与内联函数

- **考法：**不建立独立文件，而是直接在脚本中定义数学函数。现代 MATLAB 推荐使用 **匿名函数句柄 (`@`)**，旧版使用 `inline` 。

- **解题步骤：**

  1. **定义句柄：**使用 `f = @(变量列表) 表达式`。
  2. **调用计算：**像普通函数一样使用 `y = f(x)` 或 `feval(f, x)`。
  3. **绘图应用：**结合 `fplot` 或 `plot` 绘制该句柄的图像。

- **示例代码：**

  ```matlab
  % --- 1. 定义函数句柄 ---
  % @(x) 表示 x 是自变量
  f_handle = @(x) sin(x) .* exp(x); 
  
  % --- 2. 调用计算 ---
  y = f_handle(0.5);      % 代入数值计算
  
  % --- 3. 绘图 ---
  fplot(f_handle, [-5, 5]); % 直接绘制句柄函数
  ```

## 数值分析 | 求根与求极值

- **考法：**利用泛函命令求解非线性方程的零点或函数的极小值点。

- **解题步骤：**

  1. **准备函数：**定义好函数句柄 `f`。
  2. **求零点：**使用 `fzero(f, x0)`，其中 `x0` 是猜测的初始点。
  3. **求极值：**使用 `fminbnd(f, x1, x2)`，在区间 `[x1, x2]` 内寻找极小值。

- **示例代码：**

  ```matlab
  f = @(x) (sin(x)).^2 - 0.5; % 定义目标函数
  
  % --- 求零点 (方程解) ---
  x_root = fzero(f, 0.5);     % 在 0.5 附近寻找零点
  
  % --- 求极小值点 ---
  x_min = fminbnd(f, 0, 1);   % 在 [0, 1] 区间找最小值位置
  ```
