# 条件判断

Go 的 `if` 类似于 C / Java，但有几点特色：

1. 可以直接在 `if` 里声明变量，作用域只在 `if` 及对应 `else` 块中。
2. 没有圆括号，条件直接写。
3. 支持 `else if`。

```go
package main

import "fmt"

func main() {
    a := 10

    // 基本 if
    if a > 5 {
        fmt.Println("a 大于 5")
    } else if a == 5 {
        fmt.Println("a 等于 5")
    } else {
        fmt.Println("a 小于 5")
    }

    // if 中声明变量
    if b := a * 2; b > 15 {
        fmt.Println("b 大于 15:", b)
    }
    // 注意：b 作用域只在这个 if 块里
}
```

> 生成一个随机数，使用if-else结构判断输入的数字与随机数的关系，并且统计猜测次数

```go
package main

import (
	"fmt"
	"math/rand"
	"time"
)

func main() {
	rand.Seed(time.Now().UnixNano()) // 以当前时间为种子
	num := rand.Intn(100)
	var guess_num int
	count := 0
	for {
		fmt.Println("请输入你猜测的数字：")
		fmt.Scanln(&guess_num)
		if guess_num > num {
			fmt.Println("太大了！")
			count++
		} else if guess_num < num {
			fmt.Println("太小了！")
			count++
		} else {
			count++
			fmt.Printf("猜对了！总共猜了%d次。", count)
			break
		}
	}
}
```

# 多分支选择

Go 的 `switch` 更加灵活：

1. 不需要 `break`，每个 case 执行完自动跳出。
2. 可以使用逗号分隔多个匹配值。
3. 支持条件 switch，而不只是匹配变量。

```go
package main

import "fmt"

func main() {
    grade := "B"

    // 普通 switch
    switch grade {
    case "A":
        fmt.Println("优秀")
    case "B", "C":
        fmt.Println("良好")
    default:
        fmt.Println("需要努力")
    }

    // 条件 switch
    score := 85
    switch {
    case score >= 90:
        fmt.Println("优秀")
    case score >= 70:
        fmt.Println("良好")
    default:
        fmt.Println("一般")
    }
}
```

> 系统巡检时根据不同的输入命令查询不同的系统状态信息并输出

在文件`check.go`中写入所有函数：

```go
package main

import (
        "fmt"
        "os/exec"
)

func check_SSD() {
        cmd := exec.Command("df", "-h")
        output, err := cmd.Output()
        if err != nil {
                fmt.Println("命令执行错误：", err)
        }
        fmt.Println(string(output))
}

func check_mem() {
        cmd := exec.Command("free", "-m")
        output, err := cmd.Output()
        if err != nil {
                fmt.Println("命令执行错误：", err)
        }
        fmt.Println(string(output))
}

func check_net() {
        cmd := exec.Command("ifconfig")
        output, err := cmd.Output()
        if err != nil {
                fmt.Println("命令执行错误：", err)
        }
        fmt.Println(string(output))
}
```

在文件`main.go`中写入执行逻辑：

```go
package main

import (
        "fmt"
)

func main() {
        var choice string
        fmt.Printf("请选择你要检查的项目：\n(a)磁盘\n(b)内存\n(c)网卡\n")
        fmt.Scanln(&choice)
        switch choice {
        case "a":
                check_SSD()
        case "b":
                check_mem()
        case "c":
                check_net()
        default:
                fmt.Println("输入选项错误！")
        }
}
```

# 循环

Go 只有一种循环结构：`for`。

- 类C的for循环

  ```go
  for i := 0; i < 5; i++ {
      fmt.Println(i)
  }
  
  ```

- 类While循环

  ```go
  i := 0
  for i < 5 {
      fmt.Println(i)
      i++
  }
  ```

- 无限循环

  ```go
  for {
      fmt.Println("无限循环")
      break // 否则会一直打印
  }
  ```

> 打印输出九九乘法表

```go
package main

import "fmt"

func main() {
	for j := 1; j <= 9; j++ {
		for i := 1; i <= j; i++ {
			fmt.Printf("%d*%d=%d  ", i, j, i*j)
		}
		fmt.Printf("\n")
	}
}
```

# 跳转

| 关键字     | 功能                        |
| ---------- | --------------------------- |
| `break`    | 跳出当前循环                |
| `continue` | 跳过本次循环，继续下一轮    |
| `goto`     | 跳转到指定标签（Go 很少用） |

```go
package main

import "fmt"

func main() {
    for i := 0; i < 5; i++ {
        if i == 2 {
            continue // 跳过 i==2
        }
        if i == 4 {
            break    // 循环结束
        }
        fmt.Println(i)
    }

    // goto 示例
    fmt.Println("goto示例开始")
    goto End
    fmt.Println("这行不会执行")
End:
    fmt.Println("跳转到 End 标签")
}
```

# 遍历

Go 提供 `range` 遍历数组、切片、map、字符串：

```go
nums := []int{10, 20, 30}
for i, v := range nums {
    fmt.Println(i, v)
}

m := map[string]int{"a": 1, "b": 2}
for k, v := range m {
    fmt.Println(k, v)
}
```

输出：

```go
//索引和元素值
0 10
1 20
2 30
//键值对
a 1
b 2
```
