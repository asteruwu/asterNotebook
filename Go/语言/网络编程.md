# net包基础

在 TCP 通信中：

- 服务端负责**监听** → **接收连接** → **通信**
- 客户端负责**发起连接** → **通信**

对应的主要接口和函数就是：

| 功能       | 函数/接口                      | 说明                                         |
| ---------- | ------------------------------ | -------------------------------------------- |
| 启动监听   | `net.Listen()`                 | 启动 TCP（或 Unix）监听，返回一个 `Listener` |
| 等待连接   | `Listener.Accept()`            | 阻塞等待，返回一个 `Conn`（连接对象）        |
| 连接服务器 | `net.Dial()`                   | 客户端连接服务器，返回一个 `Conn`            |
| 通信       | `Conn.Read()` / `Conn.Write()` | 通过连接读写数据                             |

## Listener

```go
ln, err := net.Listen("tcp", ":8080")
```

这行代码会：

- 在本地 `8080` 端口上启动一个 TCP 监听；
- 返回一个实现了 `net.Listener` 接口的对象。

主要方法：

```go
type Listener interface {
    Accept() (Conn, error)   // 接收一个新连接
    Close() error            // 关闭监听
    Addr() Addr              // 获取监听地址
}
```

> [!IMPORTANT]
>
> `ln.Accept()` 会**阻塞**，直到有客户端连接；每当有连接（调用`ln.Accept()`）到来时，就返回一个新的 `Conn` 对象，常配合goroutine并发处理多个客户端。

## Conn

`Conn` 是客户端与服务端之间的**双向数据流通道**。

核心方法：

```go
type Conn interface {
    Read(b []byte) (n int, err error)   // 从连接中读取数据
    Write(b []byte) (n int, err error)  // 向连接写数据
    Close() error                       // 关闭连接
    LocalAddr() Addr                    // 本地地址
    RemoteAddr() Addr                   // 对端地址
}
```

## Dial

`net.Dial()` 是客户端发起连接的入口。

```go
conn, err := net.Dial("tcp", "127.0.0.1:8080")
```

参数说明：

- 第一个参数是网络类型（`tcp` / `udp` / `unix`）
- 第二个参数是目标地址（`ip:port`）

`Dial` 会返回一个 `Conn`，后续客户端也可直接 `Read` / `Write`。

## 示例

> Go TCP 并发客户端/服务端

### server.go

- **创建监听器**

  ```go
  ln, err := net.Listen("tcp", ":8080")
  ```

  - `net.Listen("tcp", addr)` 返回一个 `Listener`，用于等待客户端连接。

  - `":8080"` 表示本机所有网卡的 8080 端口。

- **Accept 接收连接**

  ```go
  conn, err := ln.Accept()
  ```

  - 阻塞等待客户端连接。

  - 返回一个 `net.Conn`，表示单个客户端的连接。

  - 可以用 `conn.RemoteAddr()` 获取客户端 IP:Port，用 `conn.LocalAddr()` 获取服务端本机地址。

- **处理连接**

  ```go
  go handle(conn)
  ```

  - 每个客户端用 goroutine 并发处理，保证服务端同时响应多个客户端。

  - 使用 `defer conn.Close()` 保证资源释放。

- **读取客户端消息**

  ```go
  msg, err := reader.ReadString('\n')
  ```

  - `bufio.NewReader(conn)` 提供缓冲读取。

  - 阻塞读取，直到收到换行符或客户端关闭连接（返回 `EOF`）。

- **响应客户端消息**

  ```go
  response := fmt.Sprintf("[来自%s] %s\n", conn.LocalAddr(), msg)
  conn.Write([]byte(response))
  ```

  - 使用 `fmt.Sprintf` 拼接格式化字符串。

  - `conn.Write` 发送数据回客户端。

- **EOF 处理**

  ```go
  if err != nil {
      fmt.Println("Client disconnected:", err)
      return
  }
  ```

  - `EOF` 表示客户端正常断开连接。

### client.go

- **发起连接**

  ```go
  conn, err := net.Dial("tcp", "localhost:8080")
  ```

  - `net.Dial` 阻塞连接服务器，返回 `net.Conn`。

- **并发多个客户端**

  ```go
  var wg sync.WaitGroup
  for i := 1; i <= 5; i++ {
      wg.Add(1)
      go client_connect(i, &wg)
  }
  wg.Wait()
  ```

  - 用 `sync.WaitGroup` 等待所有 goroutine 完成，避免主程序提前退出。

  - 每个客户端 goroutine 独立运行，互不干扰。

- **发送消息**

  ```go
  text := fmt.Sprintf("Client %d input: %d\n", i, j)
  conn.Write([]byte(text))
  ```

  - 自动生成消息内容。

  - 不依赖终端输入，适合批量并发测试。

- **接收服务端响应**

  ```go
  reply := make([]byte, 1024)
  n, _ := conn.Read(reply)
  fmt.Printf("Client %d received: %s", i, string(reply[:n]))
  ```

  - 为接收数据创建缓冲区。

  - `Read` 阻塞直到服务端返回消息。

  - 只读取实际字节数 `n`，并转为字符串打印。

### 完整代码（附带注释）

`server.go`

```go
package main

import (
	"bufio"
	"fmt"
	"net"
)

func main() {
	ln, err := net.Listen("tcp", ":8080") //创建一个listener,监听8080端口
	if err != nil {
		panic(err)
	}
	fmt.Println("Server is listening on :8080")

	//持续等待新的连接，Accept后返回信息交流的通道conn，并执行相应逻辑
	for {
		conn, err := ln.Accept()
		if err != nil {
			fmt.Println("Accept error:", err)
			continue
		}
		fmt.Println("[连接]新的客户端接入：", conn.RemoteAddr())
		go handle(conn)
		fmt.Println("[断开]客户端离开：", conn.RemoteAddr())
	}
}

// 定义具体执行逻辑，针对传入的通道conn，读取客户端发送的信息并重新写入，带上服务端本机地址并响应
func handle(conn net.Conn) {
	defer conn.Close()
	reader := bufio.NewReader(conn)

	for {
		msg, err := reader.ReadString('\n')
		local_addr := conn.LocalAddr().String()
		response := fmt.Sprintf("[来自%s] %s\n", local_addr, msg)
		if err != nil {
			fmt.Println("Client disconnected:", err)
			return
		}
		conn.Write([]byte(response))
	}
}
```

`client.go`

```go
package main

import (
	"fmt"
	"net"
	"sync"
)

func client_connect(i int, wg *sync.WaitGroup) {
	defer wg.Done()
	conn, err := net.Dial("tcp", "localhost:8080") //向地址localhost:8080发起tcp连接，并得到通道conn
	if err != nil {
		panic(err)
	}
	defer conn.Close()

	for j := 1; j <= 3; j++ {
		text := fmt.Sprintf("Client %d input: %d\n", i, j)         //自动生成信息并用变量text存储
		conn.Write([]byte(text))                                   //将text的信息写入通道conn
		reply := make([]byte, 1024)                                //定义变量reply空字节切片存储从服务端返回的信息
		n, _ := conn.Read(reply)                                   //接收从通道conn获得的信息
		fmt.Printf("Client %d received: %s", i, string(reply[:n])) //将信息写入reply并转为字符串，然后在客户端打印
	}
}

func main() {
	var wg sync.WaitGroup
	for i := 1; i <= 5; i++ {
		wg.Add(1)
		go client_connect(i, &wg)
	}

	wg.Wait()
}
```

# net/http

| 概念                                 | 说明                 |
| ------------------------------------ | -------------------- |
| `http.ListenAndServe(addr, handler)` | 启动一个 HTTP 服务器 |
| `http.Handler`                       | 处理 HTTP 请求的接口 |
| `http.HandleFunc(path, func)`        | 注册路由与处理函数   |
| `http.ResponseWriter`                | 向客户端写响应       |
| `*http.Request`                      | 表示一次客户端请求   |

> 最小http服务，根据地址内容进行输出

```go
package main

import (
	"fmt"
	"net/http"
)

// 定义客户端请求路由/user时服务器的执行逻辑
func httpHello(w http.ResponseWriter, r *http.Request) {
	fmt.Fprintf(w, "Hello: %s\n", r.URL.Query().Get("name")) //从URL参数中取出name值
}

func main() {
	http.HandleFunc("/user", httpHello)      //确立路由与相关逻辑的映射关系
	err := http.ListenAndServe(":8080", nil) //启动一个http服务器，监听8080端口
	if err != nil {
		fmt.Println("Server error:", err)
	}
}
```

启动后访问`http://localhost:8080/user?name=aster`，客户端返回`Hello: aster`。

## Server

- **启动 HTTP 服务**

  - `http.ListenAndServe(addr string, handler http.Handler)`

    - 启动一个 HTTP 服务器，监听指定的端口和地址。
    - `addr` 是监听的地址和端口，比如 `":8080"` 表示监听本地 8080 端口。

    ```go
    http.ListenAndServe(":8080", nil)  // 启动服务器，监听 8080 端口
    ```

- **路由与请求处理**

  - `http.HandleFunc(pattern string, handler func(http.ResponseWriter, *http.Request))`

    - 用于注册一个 URL 路由模式和一个处理函数。当用户访问该 URL 时，指定的处理函数会被调用。

    ```go
    http.HandleFunc("/hello", func(w http.ResponseWriter, r *http.Request) {
        fmt.Fprintln(w, "Hello, World!")
    })
    ```

    > [!NOTE]
    >
    > 处理函数的参数必须是`http.ResponseWriter`和`*http.Request`。

- **HTTP 请求与响应**

  - `http.Request`

    - 用于描述 HTTP 请求的结构体，包含了请求的 URL、方法（GET、POST等）、请求头、请求体等信息。
    - 你可以通过 `r.Method`, `r.URL`, `r.Header` 等字段访问请求的具体信息。

  - `http.ResponseWriter`

    - 用于构建并发送 HTTP 响应给客户端。通过它，你可以设置响应状态码、响应头以及响应体。

    - 常用方法：

      - `w.Write([]byte)`：写入响应体内容。
      - `w.WriteHeader(statusCode)`：设置响应状态码（默认是 200 OK）。

      ```go
      func handler(w http.ResponseWriter, r *http.Request) {
          w.Header().Set("Content-Type", "text/plain")
          w.WriteHeader(http.StatusOK)  // 设置响应状态码
          w.Write([]byte("Response content"))  // 响应内容
      }
      ```

## Client

- **发起HTTP请求**

  - `http.Get(url string)`：发起一个 GET 请求。
  - `http.Post(url, contentType string, body io.Reader)`：发起一个 POST 请求，常用于发送表单数据或者 JSON 数据。
  - `http.NewRequest(method, url string, body io.Reader)`：可以创建一个自定义 HTTP 请求，支持自定义请求方法、请求体、请求头等。

- **设置 HTTP 请求头**

  - 通过 `http.Request` 的 `Header` 字段设置 HTTP 请求头。`Header` 是一个 `map[string][]string` 类型。

  ```go
  req, err := http.NewRequest("GET", "http://example.com", nil)
  req.Header.Set("Authorization", "Bearer token")
  ```

## 示例

```go
package main

import (
	"encoding/json"
	"fmt"
	"net/http"
	"runtime"
)

//定义存储系统信息的结构体
type Info struct {
	CpuCount     int `json:"cpu_count"`
	GoroutineNum int `json:"goroutines_num"`
	MemAlloc     int `json:"mem_alloc"`
	MemSys       int `json:"mem_sys"`
}

//生成runtime数据，并赋值给结构体
func runtime_info(i *Info) {
	i.CpuCount = runtime.NumCPU()
	i.GoroutineNum = runtime.NumGoroutine()

	var mem runtime.MemStats
	runtime.ReadMemStats(&mem)
	i.MemAlloc = int(mem.Alloc) / 1024
	i.MemSys = int(mem.Sys) / 1024
}

//定义基于结构体的响应处理函数
func (i Info) check_runtime_system(w http.ResponseWriter, r *http.Request) {
	runtime_info(&i)

	w.Header().Set("Content-Type", "application/json")
	json.NewEncoder(w).Encode(i)
}

func main() {
	sysInfo := &Info{}

	http.HandleFunc("/syscheck", sysInfo.check_runtime_system)
	fmt.Println("正在启动服务器，监听8080端口……")
	http.ListenAndServe(":8080", nil)
}
```

# RESTful API

**REST**（Representational State Transfer）是一种架构风格，强调客户端与服务器之间的无状态通信。它使用标准的 HTTP 方法（如 GET、POST、PUT、DELETE）进行交互，通常是基于资源的操作。

- **资源**：每个 URL 表示一种资源。

- **HTTP 动词**：与资源操作相关的标准 HTTP 方法（如 GET 查询、POST 创建、PUT 更新、DELETE 删除）。

- **无状态性**：每个请求都应包含足够的上下文信息，服务器不应保存客户端状态。

- **统一接口**：API 应遵循一定的约定，使得所有请求结构一致，易于理解和维护。

> 示例

```
GET /users                 // 获取所有用户
GET /users/{id}            // 获取指定 ID 的用户
POST /users                // 创建新用户
PUT /users/{id}            // 更新指定 ID 的用户
DELETE /users/{id}         // 删除指定 ID 的用户
```

# JSON编解码

**JSON (JavaScript Object Notation)** 是一种轻量级的数据交换格式，具有良好的可读性，广泛用于 Web API 数据传输。

- **JSON 结构**：由键值对（key-value pairs）组成，键（key）是字符串类型，值（value）可以是字符串、数字、布尔值、数组、对象或 `null`。

- **编解码方法：**

  - **编码** | go对象 $$\rightarrow$$ JSON

    使用 `json.Marshal()` 将 Go 对象转换为 JSON 字符串。

    ```go
    package main
    
    import (
        "encoding/json"
        "fmt"
    )
    
    type User struct {
        Name string `json:"name"`
        Age  int    `json:"age"`
    }
    
    func main() {
        user := User{"Alice", 25}
        jsonData, err := json.Marshal(user)
        if err != nil {
            fmt.Println("Error:", err)
        }
        fmt.Println(string(jsonData))  // 输出：{"name":"Alice","age":25}
    }
    ```

  - **解码** | JSON $$\rightarrow$$ go对象

    使用 `json.Unmarshal()` 将 JSON 字符串解析为 Go 对象。

    ```go
    package main
    
    import (
        "encoding/json"
        "fmt"
    )
    
    type User struct {
        Name string `json:"name"`
        Age  int    `json:"age"`
    }
    
    func main() {
        jsonData := `{"name":"Bob","age":30}`
        var user User
        err := json.Unmarshal([]byte(jsonData), &user)
        if err != nil {
            fmt.Println("Error:", err)
        }
        fmt.Println(user)  // 输出：{Bob 30}
    }
    ```
